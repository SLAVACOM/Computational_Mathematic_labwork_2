# Contributing

Спасибо за интерес к проекту.

## Как предложить изменения

1. Сделайте форк или создайте ветку от `main`.
2. Назовите ветку по смыслу, например: `feature/new-system` или `fix/newton-stop-criteria`.
3. Внесите изменения небольшими логичными коммитами.
4. Откройте Pull Request с кратким описанием: что изменено и зачем.

## Минимальные правила

- Соблюдайте читаемость кода и понятные имена.
- Не смешивайте в одном PR несвязанные изменения.
- Если меняется поведение алгоритма, добавьте пояснение в `README.md` или `CHANGELOG.md`.

## Рекомендации по сообщениям коммитов

Используйте короткий формат:

- `feat: add new 3d nonlinear system`
- `fix: improve simple iteration stopping condition`
- `docs: update quick start section`
