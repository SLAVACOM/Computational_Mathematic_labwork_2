# Code of Conduct

Этот учебный репозиторий поддерживает уважительное и профессиональное общение.

## Наши принципы

- Вежливость и конструктивная обратная связь.
- Уважение к различным мнениям и уровню подготовки участников.
- Фокус на техническом обсуждении, а не на личности.

## Неприемлемое поведение

- Оскорбления, травля, дискриминационные высказывания.
- Агрессивная или токсичная коммуникация.
- Публикация личной информации без согласия.

## Контакты

Если возникла проблема с коммуникацией, создайте приватное обращение владельцу репозитория (`slava`) через удобный канал.
